Let me fix all and for the moment try it if you like how it is going on

This OCX and sample project REQUIRES:
1) Directx8 or above installed
2) Win9x/WinMe/Win2000 (not NT)
3) 3D Video Card
4) Visual Basic 6 runtime Installed
Without a 3D video card the program will run at REF mode, using software algorithms bypassing any 3D accelleration. You will be able to see the whole features but the performane will be really low!

Features:
- Add Frames and meshes separatelly controlling them as single or grouped
- Mouse or keyboard Movement
- Tree object scene viewer (now it is not that useful but i am working on)
- Walking mode
- Picking mode
- DXF loader
- X file loader
- Extruded mesh builder
- Land/Ground mesh generator (see Terrain.ocx)
- Easy Textures
- Space world simulation
- Add Material/Textures opaque or transparent
- Easy load custom data file
